Allen Disk

強大、免費、開源的雲端硬碟，你可以擁有自己的網路硬碟，也開放註冊帳號，當作免空。 Allen Disk 支援檔案加密、外連檔案、線上預覽、分享資料夾等功能，讓你輕鬆管理或分享你的檔案。

功能特色
使用介面
採用Bootstrap設計，介面簡約，一目了然，操作簡便
快速分享
強大的分享功能，支援分享單一檔案或資料夾，可線上查看或下載
線上預覽
Allen Disk 支援在線上預覽影音、文字檔、圖片，甚至網頁、 Microsoft Office 文件
簡單上傳
Allen Disk 的上傳介面簡潔易懂，支援拖拉上傳、選取檔案上傳、遠端下載、甚至上傳資料夾
加密儲存
Allen Disk 從第一版起即支援加密儲存用戶檔案，從 1.5 版起更是強化了加密系統，使檔案不被外人輕易瀏覽
完全免費
更重要的是， Allen Disk 採用 MIT 授權釋出，不論個人、團體、商業使用，都是完全免費的，同時也允許任何人對主程式做出修改

安裝教學

系統需求 （安裝時會替您檢測）

PHP 5.2以上
MySQL 5.0以上
PHP模組 Mysqli, MCrypt, Multibyte String

自動安裝

  1. 上傳檔案到FTP
  2. 打開 Allen Disk 網址
  3. 確認所有安裝條件符合
  4. 填入 MySQL 資訊
  5. 填入網站資訊
  6. 新增一個帳號

手動安裝

若無法透過安裝程序進行安裝，請使用手動安裝。

  1. 打開 /database.php 並依照註解填入資料
  2. 將 /install/install.sql 匯入 MySQL
  3. 到設定網址 /admin 填入網站資訊並新增帳號
管理後台預設密碼為 1234

自動升級（只能從 1.4 版升級）

  1. 備份 /database.php , /file 與資料庫內容
  2. 上傳新版 Allen Disk ，保留 /database.php , /file
  3. 打開 Allen Disk 網址
  4. 確認所有安裝條件符合
  5. 選擇「從 1.4 升級」

還是不了解？看看圖文教學吧！
